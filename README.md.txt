Retail Dashboard Project
Overview

This project analyzes retail sales data using an interactive dashboard built in Google Data Studio (Looker Studio).
It focuses on understanding sales performance, customer behavior, and revenue trends to support data-driven decision-making.

Key Insights
Clothing dominates overall sales volume across categories
Revenue is relatively consistent across different age groups
Revenue shows a steady growth trend over time, indicating stable business performance
Tools Used
Google Data Studio (Looker Studio)
Data visualization and dashboard creation
Basic data cleaning and transformation
Project Contents
Dataset (Retail Sales Data)
Dashboard Screenshot
Presentation (PPT – 6 slides)
Objective

To analyze retail transaction data and identify key business insights that help improve revenue understanding, category performance, and overall sales strategy.

Outcome

The dashboard provides a clear visual summary of business performance and highlights key revenue drivers, enabling better strategic decision-making.
